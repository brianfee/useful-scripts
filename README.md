# useful-scripts

## PDF-Protect
### Requirements
- bash
- pdftk

### Purpose
- Creates a protected copy of a pdf with (Protected) appended to the name of the file.

### Usage
<pre>
pdf-protect.sh <i>file_name</i>
</pre>


## lscsv.sh
### Requirements
- bash
- python

### Purpose
- Export a list of files in current directory to a file. Default: *filelist.csv*

### Usage
<pre>
lscsv.sh
</pre>
